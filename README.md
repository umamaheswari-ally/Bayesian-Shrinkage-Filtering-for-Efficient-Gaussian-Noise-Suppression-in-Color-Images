# Bayesian Shrinkage Filtering for Efficient Gaussian Noise Suppression in Color Images

## Overview
This project implements Bayesian Shrinkage Filtering to remove Gaussian noise from color images.

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook

## Features
- Noise reduction in color images
- Image enhancement
- Performance evaluation
